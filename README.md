# travel-bucket-list
A web application designed to help users organize travel destinations, categorize trips, and track progress toward personal travel goals.

Open index.html directly in your browser — no build step, no dependencies, no server required.

Key Features
Interactive Goal Tracking — Visual progress bar shows visited vs. planned destinations at a glance.

Destination Management — Add, filter by status or category, toggle visited, and remove destinations.

Category Labels — Organize trips by City, Nature, Beach, Adventure, Culture, or Other.

Optional Notes — Attach a personal note to each destination (best season, things to do, etc.).

Local Persistence — All data is saved to localStorage so your list survives page refreshes.

Responsive Design — Works on desktop, tablet, and mobile without any framework.

Get started:
git clone https://github.com/srathira/travel-bucket-list.git
cd travel-bucket-list
open index.html   # macOS
or double-click index.html on Windows/Linux

# Features Walkthrough-
Add a Destination
Fill in the place name, pick a category, and optionally add a note. Press + Add or hit Enter.

Mark as Visited
Click the circle checkbox on the left of any card, or use the ✓ button on the right. The card moves to "Visited" styling automatically.

Filter Your List
Use the filter buttons to view All, Planned, or Visited destinations or narrow by category.

Progress Tracking
The header progress bar updates live as you mark destinations visited, showing your completion percentage.
